Getting Started

1. Include Smooth Scroll on your site.

<script src="smooth-scroll.js"></script>

2. Add the markup to your HTML.

Turn anchor links into Smooth Scroll links by adding the .scroll class. Give the anchor location an ID just like you normally would.

<a class="scroll" data-speed="500" data-easing="easeInOutCubic" data-url="false" href="#bazinga">Anchor Link</a>
...
<span id="bazinga">Bazinga!</span>

New Fixed Headers

Add a .scroll-header class to fixed headers. Smooth Scroll will automatically offset scroll distances by the header height.

<nav class="scroll-header">
	...
</nav>

New Updating URLs

Use the optional data-url attribute to upate the URL with an anchor link. It accepts true and false values, and defaults to false if you don't specify.

New Easing with Ease

Use optional data-speed and data-easing attributes to control the speed and easing of your animations. If you don't specify, Smooth Scroll defaults to 500 milliseconds and easeInOutCubic easing.

Easing Options

You can set both the animation style and the rate of acceleration. Not sure what these names mean? Learn more at easings.net.

Linear

  • linear

Ease-In

  • easeInQuad
  • easeInCubic
  • easeInQuart
  • easeInQuint

Ease-In-Out

  • easeInOutQuad
  • easeInOutCubic
  • easeInOutQuart
  • easeInOutQuint

Ease-Out

  • easeOutQuad
  • easeOutCubic
  • easeOutQuart
  • easeOutQuint
  • Linear: Moves at the same speed from start to finish.
  • Ease-In: Gradually increases in speed.
  • Ease-In-Out: Gradually increases in speed, peaks, and then gradually slows down.
  • Ease-Out: Gradually decreases in speed.

3. Pat yourself on the back.

Seriously, you're done. Nice work! Modify as needed.

Browser Compatability

Smooth Scroll works in all modern browsers, including IE 9 and above.

In older browsers, anchor links jump the way they normally would. If you need to smooth scrolling for older browsers, download the jQuery version of Smooth Scroll on GitHub.
